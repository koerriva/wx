## Dependencies Library
+ OpenGL4.5
+ freetype2
+ glfw3
+ glm
+ spdlog

## Description

this software aim to build an Open World Action - RPG game about east style.
being early development.

## Build
env:`Mingw64`
```
    mkdir build & cd build
    cmake .. -v -j8
```
